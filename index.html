Ресурспаки

Better Leaves

Объёмный вид листвы для более реалистичного внешнего вида деревьев. Работает при настройке графики на "Детализированные" для листвы.

Low On Fire

Делает эффект горения на экране менее навязчивым и раздражающим, сохраняя при этом важную информацию о состоянии.

Redstone Tweaks

Улучшенный внешний вид редстоуновых блоков с полезными индикаторами, например, стрелки для направления воронок.

Балансировка торговли

Экспериментальная функция

Балансировка торговли — это эксперимент для Java Edition и Bedrock Edition, который меняет сделки библиотекаря и странствующего торговца для более честной торговли. Требует включения в настройках мира.

Инструкция для TLauncher

1

Открой TLauncher

Запусти TLauncher и нажми на кнопку "TLMods"

2

Создай сборку

Нажми "+Создать", введи название "Chinazes"(можно любое), выбери Fabric и версию 1.21.8

3

Первый запуск

Нажми "+Создать", затем запусти Minecraft для создания папок

4

Скачай и установи

Скачай полную сборку ниже, распакуй и перемести файлы с заменой в:
C:\Users\USER\AppData\Roaming\.minecraft\versions\ТВОЯ_СБОРКА\

5

Готово!

Запускай игру через свою сборку и получай вайбик!
